Here´s some stuff about me:

I'm a Computer Science student at UCM in Spain. I focus on system architecture and software engineering, with the aim of building clean, reusable, and rock-solid codebases and tools. When I'm not doing schoolwork, I build tools for my colleagues to use, in addition to participating in Game Jams with my team.

Currently, my weapons of choice are Unity3D and C# for game development, and Java with InteliJ for tools and general application development. I also dabble in C++, using CMake as my build system.

I use Git with Commander for Windows, and pure old-school console Git with GNU/Linux.
